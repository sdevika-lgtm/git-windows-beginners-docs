Beginner-friendly documentation for setting up Git on Windows using Git Bash and Visual Studio Code.

## Overview
This project contains a set of beginner-focused documents designed to help new users install Git, understand the basic workflow, and start working with Git locally on a Windows system.

## Key Features
- Step-by-step beginner guides
- Visual Git command examples
- Real-world Git errors and fixes
- Merge conflict and recovery examples
- Docs-as-code portfolio project

## Who This Project Is For
The project is intended for:
- beginners who are new to Git
- users learning Git Bash on Windows
- aspiring technical writers exploring docs-as-code
- anyone who wants step-by-step Git documentation without unnecessary jargon

## What This Project Covers
- installing Git on Windows
- opening and using Git Bash
- configuring Git with username and email
- creating a local repository
- understanding beginner Git concepts
- resolving common beginner confusion

## Tools Used
- Git
- Git Bash
- Visual Studio Code
- Markdown

## Project Goal
The goal of this repository is to provide clear, beginner-friendly Git documentation while also serving as a docs-as-code portfolio project.

## Notes
This project currently focuses on **local Git workflows and fundamentals**.

Future improvements may include:
